# Travian Scripts

These userscripts were experimental and are no longer maintained for public use.

Using automation scripts in Travian may violate the game's rules and can result in account bans, especially when scripts are outdated or behave differently from the current game client. For that reason, the functional code has been removed from this public repository.

If you still want to take that risk, please open an issue in this repository and explain what you need.
